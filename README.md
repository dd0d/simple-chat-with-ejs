# Simple Chat with EJS

A lightweight and easy-to-use chat application built with **Node.js**, **Express**, **MongoDB**, and **EJS**.  
This app allows users to register, log in, and chat in real-time with other users in a clean and simple interface.

## Features
- User authentication: Register and login securely
- Real-time messaging
- Search and chat with specific users
- Simple and clean EJS templates
- MongoDB for storing users and messages

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dd0d/simple-chat-with-ejs.git
````

2. Install dependencies:

   ```bash
   npm install
   ```
3. Create a `.env` file and add your MongoDB URI:

   ```env
   MONGO_URI=your_mongodb_uri
   ```
4. Run the application:

   ```bash
   npm start
   ```
5. Open your browser at `http://localhost:3000`

## Technologies Used

* Node.js
* Express.js
* MongoDB
* EJS (Embedded JavaScript Templates)
* Socket.io (for real-time chat)
